bastian void / ant'lrd japan tour

日本ツアー 2017 情報

アメリカから来日する二人の音楽家、バスティアン・ヴォイドとアントルドが、ぜひ会いたいと熱望した音楽家と 東京・金沢・京都・大阪 にて計6つの公演をお送りします。
国境を越えてつながる現行の地下実験音楽シーンを象徴する素敵なラインナップが揃いました。どうぞご期待ください。


Bastian Void

1986年生まれ、マサチューセッツ州出身のグラフィックデザイナー・作曲家として活躍するジョセフ・バスタードが手掛ける音楽プロジェクト。
2009年にLooks Realisticでのバンド活動を始め、2011年に初めてのBastian Voidとしてのアルバム「Ported」をリリース。それ以降、バスチャン・ヴォイドなどの名義で、Orange Milk、birdFriend、Digitalis、Constellation Tatsu、Phineryなどのレーベルで数多くのアルバムをリリース。作曲の際、アナログ、デジタル、モジュラーシンセサイザーを使用。Moss Archiveというレーベルを経営している。

NEW ALBUM 'ADAPTER' COMING IN OCTOBER ON BIRDFRIEND!

Ant'lrd

ポートランド在住のサウンド・アーティスト、パーカッショニスト、Colin Blantonによるソニック・プロジェクション。
Moss Archiveから最新作Cherubianをリリース。過去には、BARO、Whited Sepulcher、Digitalis、Sacred Phrases、Inner Islandsなどのレーベルから、いくつかのカセットやLPをリリースしている。

Open 6:30 pm / Start 7 pm | entrance fee 3000 yen (+ 1drink order)

アメリカツアーの際にはbastian voidの家に泊めさせてもらったというほどの仲良し、食品まつり a.k.a Foodman がソロ演奏 ＆ bastian voidとのセッションを予定。
その他の共演は、同時期に日本をツアーする二人のカナダ人アンビエント音楽家、Daniel MajerとJonathan William Scherk(ジョナサンはこの日が誕生日！)。


食品まつり a.k.a Foodman

名古屋出身、横浜在住、トラックメイカー / 絵描き。
2012年にOrange Milkよりリリースした『Shokuhin』でデビューを果たす。Juke/footwork,ambient,new age,collageなどをミックスした作風で、七尾旅人、禁断の多数決、NATURE DANGER GANG、泉まくら、DOTAMAなどのリミックス、トラック制作も手がける。

H.Takahashi

1986年 東京生まれ。建築家・作曲家。
DIYを主軸にできる限り要素を減らした簡素な方法で制作を行う。作曲にはiPhoneのGarageBandを用いアンビエントをメインとした作品をWhere To Now?(UK)、Constellation Tatsu(US)等々から発表。
また、アンビエントユニットUnknown Meの作曲なども行う。

Open / Start 7 pm | entrance fee 2400 yen (+ 1drink order)

共演は、bastian voidが運営するカセットレーベル「Moss Archive」から新作をリリースした奈良在住のカナダ人音楽家Enduranceなど。
また、Enduranceが運営するレーベル「Muzan Editions」より bastian void / ant'lrd のスプリットカセットを発売。


Takashi Hattori

大阪在住の音楽家。映像作品も手がける。映画美学校音楽美学講座の第一期生。当時まだ15 歳だったにもかかわらず、特別に入学を許可される。
2013年12月、6曲入りの初作品集『UNBORN』を円盤レコードより発表。2015年11月、サウンドトラック兼自身初のフルアルバム『MOON』を nobleより発表。楽曲提供やバンド参加などなど ・現在新作制作中！

Takahiro Mukai

日本の作曲家、大阪市生まれ大阪市在住。一般的な会社員(妻と1女扶養)。
現在は小型アナログマシンをメインとして8ステップの電子音をただただ放つのがスタイル。
ちなみに小学六年時の修学旅行先はアマゾン川支流（ペルー北部）。

7FO

大阪在住の音響作家。アンビエント、電子音楽、ダブ、サイケデリック、辺境音楽などに影響を受け、2010年、制作を開始。
ギター音の音響加工をベースに、サンプラー、シンセ、ジャンク機材等を組み合わせて宅録、ライブ。そのサウンドは、煌びやかな持続音や発振音が、夢想的かつユーモラスに、無機/電子の世界や精神世界を漂いながら、意識の中に寓話を描くように響く。
2016年にNYCのRVNG Intl.からアルバム『Water Falls Into A Blank』をリリース。
